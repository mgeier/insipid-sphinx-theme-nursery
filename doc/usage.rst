``html_theme_options``

Settings inherited from the ``basic`` theme (like most Sphinx themes)::

    html_theme_options = {
        'nosidebar': True,
        'sidebarwidth': 230,
        'body_min_width': 450,
        'body_max_width': 800,  # None to disable
        'navigation_with_keys': True,  # Use left and right arrow keys to turn pages
    }
